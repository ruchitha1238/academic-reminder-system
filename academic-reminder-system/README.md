# 🎓 Academic Reminder System

## 🚀 Problem Statement

Many students forget exam and assignment deadlines, which creates stress and last-minute pressure.

## 💡 Solution

I built a simple automated reminder system that:

- Allows users to add exam or assignment details
- Stores data using Webhooks
- Automatically checks upcoming deadlines daily
- Sends email reminders one day before
- Updates reminder status to avoid duplicate emails

## 🛠 Tech Stack

- n8n (Workflow Automation)
- Webhooks
- Google Sheets (Database)
- Gmail API
- HTML, CSS, JavaScript

## 🔁 How It Works

1. User submits reminder from frontend
2. Data is stored in Google Sheets
3. Schedule Trigger runs daily
4. Email is sent automatically
5. reminder_sent column updates to "Yes"
